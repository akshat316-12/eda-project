# Netflix Titles EDA

This project performs Exploratory Data Analysis (EDA) on the Netflix Titles dataset to understand patterns in Netflix’s content catalog.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Dataset overview
- Missing value handling
- Movies vs TV Shows distribution
- Top countries producing content
- Content growth over time
- Content rating distribution
- Top genres analysis

## Key Insights
- Movies dominate Netflix’s catalog compared to TV shows.
- The United States produces the most Netflix content.
- Netflix content increased significantly after 2015.
- TV-MA and TV-14 are the most common ratings.
- Drama and International Movies are the most common genres.

## Dataset
Netflix Movies and TV Shows dataset from Kaggle.
